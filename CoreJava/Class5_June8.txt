Java Naming Conventions:
------------------------

package		java.lang,java.io,java.util,..			All packages are prepared in lowercase & separated by . operator

class		System,String,Math,FileWriter,..		All classes are prepared each word first letter will be capital

method		println(),main(),isEmpty(),toLowerCase()	All methods are prepared each word first letter will be capital except the 1st word

constant	PI,MIN_VALUE,MIN_PRIORITY			All contants are prepared in uppercase & separated by _ operator

varaible	out,in,err,length				All variables are prepared each word first letter will be capital except the 1st word



Example:
		A_B_C
		appleIsFruit
		MangoIsFruit
		orangeIsInGreen()
		today.is.sunday

Java Tokens:
============
	Token is a piece of java code it can be anything as given below.

Keywords		=	import	,class, public, static,void,double,if,else
Identifiers		=	AreaOfCircle,radius,result,args
Literals		=	4.5,0,"Radius cannot be negative"
Variables		=	radius,result,args
Constant		=	PI
Datatype		=	double
Operators		=	=,<,*,(),{},[],;,.
Control Statements	=	if,else
