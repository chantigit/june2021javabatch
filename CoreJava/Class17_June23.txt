Constructor:
===========
Def:
	It is used to Construct the object by supplying the values

Features:

	1.Name 			->	Same as class name
	2.Access modifier	->	public
	3.Return Type		->	no
	4.Return Value		->	Current class object
	5.Invoke		->	At the time of object creation
	6.Overload		->	Yes

Types:
	1.No arg constructor  / Zero -arg constructor  / Non- paramterlized constructor 
	2.All arg constructor / Paramterlized constructor 

Note:
	For every java class, compiler supplies 1 Non- paramterlized constructor i.e called as DEFAULT CONSTUCTOR.
	DEFAULT CONSTUCTOR always supplies DEFAULT VALUES
