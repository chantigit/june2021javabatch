3.How many types of tags are there in html ?

	1.Paired tags			:	These tags has open & closed syntax
				Ex:			<p>		Text		</p>	
							<b>   	Text		</b>
	2.Unpaired tags			:	These tags has open syntax
				Ex:			<br>,<hr>,<input>,....
	3.Self closing tags		: Similar to Paired tags bute can be closed at end og open tag.
				Ex:			<img />
